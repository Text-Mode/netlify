ListGPS

Description

ListGPS is a command line tool for bulk extracting GPS co-ordinates from jpeg images as either a CSV file, or a KML data (for importing into Google Maps). The jpeg images must have been saved with location data enabled to be able to extract GPS information.

Examples

listgps input.jpg

listgps *.jpg > output.csv

listgps *.jpg output.kml

Importing a KML file into Google Maps
