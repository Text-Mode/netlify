2.7 Release
  • Internal ICO library for decoding and encoding Windows Icon files (1, 4, 8, and 24-bits).
  • Support Netscape GIF looping extension.

2.6 Release
  • Support for a Barcode plugin for reading QR-codes (based on the ZBar library).
  • Option to delete images to the Recycle bin.
  • Can toggle the colour for the transparent layer of an image (T).
  • Grayscale and negative colour commands.
  • Limited Undo support (Ctrl+Z).

  • Saving and loading of uncompressed bitmap (with alpha) file format (*.bgra).
  • Converts alpha transparency from GIF and PNG files to WEBP when saving.
  • Extended EXIF (including GPS) and image meta information (Shift+F10).

  • Fixes and performance improvements for GIF decoding.
  • Decodes 2-bit PNGs correctly.

2.1 Release

  • Use the libpng library to decode PNG images (2x faster).
2.0 Release

    (left=previous image, right=next image, up=full screen, up-down=refresh, down-right=close, diagonal-up-right=zoom in, diagonal-down-left=zoom out).

  • Added CTRL+F5 to refresh whole file list.

1.1 Release
  • Added support for clipboard copy and paste operations.

  • Can preform zoom on the current image by holding CTRL and scrolling the mouse wheel.
1.0 Release
  • Can save images as Webp format. To set the default quality set the following registry DWORD value between 0 and 100:

    HKEY_CURRENT_USER\Software\WebpImageViewer\Webp Image Viewer\Settings\WebpQuality

  • Uses the FreeImage.DLL to open and save additional image types.
  • Added an internal decoder for PNG images.
  • File image lists are now sorted numerically.
0.98 beta
  • Added support for the RIOT export plugin.
  • Added the file rename command (F2).
  • Added the verbs COPY and MOVE for the external commands menu, to copy or move the current image to a specified folder, i.e "COPY C:\MyImages"
  • Added a menu option to disable joystick integration.
0.97 beta
  • Added a Refresh current file command (F5).
  • Added the Assoicate Extension command to associate the viewer with the .webp extension.
  • Using the JPEG lossless 360 degree rotation will also clear any embedded EXIF information.
  • Added the external application menu. To configure the external application set the following registry key to the executable:

    HKEY_CURRENT_USER\Software\WebpImageViewer\Webp Image Viewer\Settings\External[1..9]

    To assign a custom title to the external application set:

    HKEY_CURRENT_USER\Software\WebpImageViewer\Webp Image Viewer\Settings\ExternalTitle[1..9]
